# 🛠️ Interactive Garden Box Configurator & Logic Engine

A professional business automation tool designed to streamline the quoting and manufacturing process for custom cedar products. This application serves as a functional logic "bridge" between specific customer requirements and real-time material calculations.

## 🚀 The Business Problem
Quoting 3-tier cedar garden boxes traditionally required 15-20 minutes of manual math to determine precise lumber dimensions, waste factors, and current market pricing. This manual bottleneck often led to inconsistent quotes and delayed customer responses.

## ✅ The Solution
I developed this interactive configurator to automate the "heavy lifting" of the business logic:
* **Dynamic Material Math:** Instantly calculates precise lumber cuts and board feet based on user-defined height, width, and depth.
* **Real-Time Pricing Engine:** A custom logic layer that updates the final quote as dimensions change, ensuring 100% accuracy.
* **User-First Interface:** Built with a focus on "speed-to-lead," allowing customers to visualize their custom build and receive a price in seconds.

## 💻 Technical Stack
* **Architecture:** Logic-first web application.
* **Frontend:** Interactive UI designed for real-time data input and visualization.
* **Deployment:** Hosted via Base44/Web environments for instant accessibility.

## 📈 Impact
This tool transforms the sales process from a manual calculation task into an automated business engine:
1. **Zero Math Errors:** Eliminates human error in material estimations.
2. **Increased Conversion:** Provides instant gratification for customers through immediate pricing.
3. **Standardized BOM:** Generates a consistent "Bill of Materials" for manufacturing, simplifying the workshop handoff.
